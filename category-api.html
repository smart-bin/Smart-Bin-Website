Nieuwe Mijlpaal API

We hebben we weer een nieuwe mijlpaal bereikt! Het is ons namelijk gelukt om data via de arduino door te sturen naar onze database via de API!
We hebben een Wifi-shield op de arduino aangesloten, een Wifi-shield maakt het mogelijk om de arduino draadloos te verbinden met het internet. Vervolgens hebben we data doorgestuurd naar onze zelfgemaakte API die het in onze database zet.
Hieronder de arduino code die we ervoor hebben gebruikt.

De data die we nu hebben kunnen doorsturen hebben we van tevoren meegegeven. De volgende stap is om de data die we van de sensor krijgen door te sturen. We hebben er alle vertrouwen in dat dit gaat lukken!

 


In de kern van het project ligt het systeem. Afval wordt gewogen en die data opgeslagen. Dit gebeurt deels via de hardware, en deels via een API.

De hardware communiceert met het internet, en stuurt wanneer nodig gegevens door naar de server, en die verwerkt het en slaat het op in een database. Dit wordt dan ontvangen door een API, die te benaderen is door iedereen.

De API kan aangeven hoeveel punten je hebt, wat de status van je bakken is, en de geschiedenis van de data van al je bakken. Deze word in een formaat genaamd JSON teruggegeven, die je via allerlei programmeertalen kan ontvangen. Een ontwikkelaar kan deze data gebruiken om een app te maken die inzicht geeft in wat je de afgelopen maand weggegooid hebt, en wanneer dit gerecycled is hoeveel er dan weer mee gemaakt zou kunnen worden. Je kunt zelfs jouw gegevens vergelijken met andere gebruikers, en zo gemiddelden opbouwen van het hele land en kijken waar jij je daarin bevind.

Een gamedeveloper kan een spel maken die jou nieuwe levens, kogels, poppetjes of in-game geld geeft. Dit geeft een gebruiker dan de mogelijkheid om zijn verzamelde punten uit te geven om allerlei spelletjes te spelen op hun favoriete platform. Je kunt, door te recycelen, direct tegen of met je vrienden spelen, en kijken wie door te recycelen de hoogste score in het spel haalt.

Alle data in de API is zo toegankelijk mogelijk gemaakt, zodat elke ontwikkelaar ermee aan de slag kan. Door onze smart-bin als platform te gebruiken voor andere ontwikkelaars zal zo een extra motief gegeven worden om afval te scheiden, en zal de populariteit hiervan omhoog gaan. Zo maken we onze wereld een stuk schoner, en is het nog leuk ook.
